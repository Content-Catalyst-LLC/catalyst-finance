# Comparison Review Checklist

- Confirm that every alternative points to the intended scenario revision.
- Confirm that at least three alternatives are included and the baseline is explicit.
- Review metric objectives and weights before using the weighted ranking.
- Treat dominance as financial-only; read every non-financial caveat.
- Confirm sensitivity paths, operations, units, and tested values.
- Confirm threshold bounds are plausible and note any multiple crossings.
- Review tornado extremes for asymmetric ranges.
- Treat one-at-a-time driver impacts as diagnostic rather than additive attribution.
- Export the comparison definition with the publication so the run can be reproduced.
