# Catalyst Finance v1.1.0

## Canonical Finance Contract and Shared Calculation Engine

This release replaces the prototype's duplicated, loosely specified calculations with one documented finance contract implemented in Python and JavaScript and verified through exact fixture parity.

### Delivered

- Versioned strict input and publication contracts
- Explicit units, bases, timing, frequency, and rounding policy
- Validated domain records and structured errors
- Stable model registry and methodology version
- Separate calculation, interpretation, and narrative layers
- Fully traceable four-component review score
- Explicit edge-case policies
- Legacy v1.0.0 migration with preservation provenance
- Python/API/browser contract parity
- Generated component and publication schemas

### Release gate

A release passes only when canonical and migrated inputs validate, generated schemas and fixtures are reproducible, Python and JavaScript outputs are equal, API validation errors are structured, and the deterministic WordPress packages pass integrity checks.
