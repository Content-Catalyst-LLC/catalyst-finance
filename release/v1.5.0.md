# Catalyst Finance v1.5.0 — Uncertainty, Monte Carlo, and Stress Testing

## Release objective

Add a transparent, seeded probability and stress-analysis layer to the canonical cash-flow engine without weakening reproducibility, review boundaries, workspace traceability, or browser parity.

## Delivered

- Versioned uncertainty definition and publication contracts.
- Uniform, triangular, normal, lognormal, and discrete distributions.
- Optional truncation and integer-valued parameters.
- Pairwise correlation with positive-semidefinite validation and Gaussian-copula Cholesky sampling.
- XorShift32 and Box–Muller random generation shared by Python and JavaScript.
- Seeded Monte Carlo analysis with rejected-draw accounting.
- Mean, median, dispersion, range, percentiles, downside probabilities, lower-tail VaR, and expected shortfall.
- Deterministic histograms and variable influence rankings.
- Multi-factor named stress cases with base-case deltas and threshold flags.
- Retained samples, reproducibility keys, schemas, examples, CLI, API, and WordPress studio.
- Append-only workspace uncertainty revisions.
- v1.4.0 scenario, comparison, and workspace migration support.

## Validation target

The release must pass Python tests, strict typing, Ruff, schema reproduction, deterministic example reproduction, Python/JavaScript uncertainty parity, PHP and JavaScript syntax, plugin packaging, repository packaging, and clean installer simulation.
