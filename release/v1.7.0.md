# Catalyst Finance v1.8.0

## Cost, Budget, and Operating Economics

This release adds a canonical operating-economics model alongside screening, capital budgeting, comparison, uncertainty, and pricing.

### Added

- Typed operating definition and publication contracts.
- Static-budget, flexible-budget, and actual operating statements.
- Sales volume, sales price, variable cost spending, fixed cost spending, and operating-profit variances.
- Positive-is-favorable variance status and exact reconciliation.
- Contribution margins, break-even units and revenue, margin of safety, operating leverage, and target-profit volume.
- Operating-unit, cost-center, and total rollups.
- JSON and CSV command-line exports.
- FastAPI evaluation and workspace revision endpoints.
- Versioned operating analyses in JSON and SQLite workspaces.
- Exact Python and JavaScript calculation parity.
- WordPress Operating Economics Studio.
- v1.6.0 migration paths for existing records and workspaces.

### Validation

The release contract covers schemas, deterministic examples, migrations, API routes, persistent revisions, browser parity, static analysis, PHP and JavaScript syntax, and deterministic repository and WordPress packages.
