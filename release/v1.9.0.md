# Catalyst Finance v1.9.0 — Evidence, Review, Governance, and Publication

Released 2026-07-17.

## Purpose

Make finance cases suitable for institutional review, responsible communication, reproducible publication, and connected platform handoffs.

## Added

- Versioned assumption, source, evidence, claim, methodology, review, attachment, and publication contracts.
- Claim-to-metric, claim-to-calculation, assumption-to-evidence, and calculation-to-source traceability.
- Append-only review requests, comments, findings, approvals, objections, revision requirements, resolutions, publication, and withdrawal events.
- Publication readiness gates that cannot be bypassed by setting a publication state.
- SHA-256 hash-chained audit history and immutable run records.
- Public redaction that excludes private records and claims dependent on private material.
- Reproducible JSON, CSV, Markdown, accessible HTML, and public read-only exports.
- Knowledge Library and Decision Studio handoff manifests with preserved identifiers.
- Governance CLI, FastAPI evaluation route, persistent workspace revisions, browser parity engine, and WordPress Governance and Publication Studio.
- Complete v1.8.0 migration coverage for all finance models and workspace records.

## Validation

The release contract runs the full test suite, Ruff, formatting, strict Mypy, schema reproduction, example reproduction, v1.8 migrations, browser parity, API smoke tests, JavaScript and PHP syntax, deterministic repository packaging, and deterministic WordPress packaging.
